### 3. Recommended Learning Path

**For Complete Beginners:**
1. Start with `01-fundamentals/01-what-is-qa/`
2. Work through Fundamentals modules (1-5)
3. Learn Manual Testing (modules 6-10)
4. Move to Automation and Techniques

**For Those with Some Experience:**
- Review Fundamentals quickly
- Focus on Testing Techniques
- Deep dive into Automation

### 4. How to Use This Repository

1. **Read the README** of each module first (conceptual understanding)
2. **Do the exercises** (hands-on practice)
3. **Study examples** (pattern recognition)
4. **Reflect** (connect to bigger picture)

See `LEARNING_GUIDE.md` for detailed guidance.
